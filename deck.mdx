</button>

```HTML

  .this-button {
    padding: 10px;

  }
</style>
<button class="this-button" onclick="scam()"}>
  Nigerian Phishing Scam
</button>
<script>
function scam() {
  alert('PWNED!!!');
}
</script>
```

## Tokens

- **Icons -** Put all the icons you're using in one place
- **Global -** This is the "CSS Reset" style that goes on the `<\body>` tag in order to standardize browser's default styling.
- **Typography -** Font family, weight, and sizes.
- **Colors -** Primary, secondary, accent colors as well as contrasts with backgrounds.
- **Breakpoints -** Traditionally, screen sizes of different devices, but you can also use math, proportions, and ratios
- **Spacing -** Rather than end up with spacing individual elements, use "small, medium, large" etc to reference actual CSS units of spacing.
- **Animations -** Common animations.
