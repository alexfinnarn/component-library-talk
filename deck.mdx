## A Bit Separated

<button style={{padding: '10px', fontSize: '18px'}}
        onClick={function(e) { e.preventDefault(); alert('PWNED!!!'); }}>
  Nigerian Phishing Scam
</button>

```HTML
<style>
  .this-button {
    padding: 10px;
    font-size: 18px;
  }
</style>
<button class="this-button" onclick="scam()"}>
  Nigerian Phishing Scam
</button>
<script>
function scam() {
  alert('PWNED!!!');
}
</script>
```

## Separation of Concerns

- Effort to reuse CSS/JS/HTML
- Makes the code look "pretty"
- "Unobtrusive JavaScript"
- Feel more like a backend, "offical" developer

## Component-driven Development

- Common to use Atomic Design: atoms, molecules, organisms, templates, and pages
- The UI is broken down into semantically-related segments
- Each segment is further broken down until you get to the atoms
